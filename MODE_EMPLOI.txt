============================================================
  PADDOCK GP - MODE D'EMPLOI
  Ton paddock personnel, facon MotoGP
============================================================

POUR LE RACE DIRECTOR (le patron)
----------------------------------
1. Ouvre l'application sur ton telephone (mets-la sur ton
   ecran d'accueil : Partager > Sur l'ecran d'accueil).
2. Touche "Race director" et entre ton code PIN.
3. "Nouveau grand prix" : nom du circuit, dates, nombre de
   jours. L'app te donne un CODE (ex : GP-LEDENON-4F7K).
4. Donne ce code aux copains du week-end. C'est tout.
5. Cree ta carte pilote comme les autres, entre tes chronos
   session par session.
6. Pendant le week-end tu peux ajouter des sessions depuis
   le panneau "Race director" en bas du live timing.
7. Dimanche soir : "Fin du GP - drapeau a damier".
   -> Podium (au cumul), resume, et TON MEILLEUR TOUR du
   week-end se grave dans ton palmares sous le circuit.
   S'il bat ton record de l'annee passee, tu le sauras.

POUR LES PILOTES INVITES
----------------------------------
1. Ouvre le lien de l'app, touche "Rejoindre un GP".
2. Entre le code donne par le race director.
3. Cree ta carte pilote : surnom, numero, ecurie, couleur,
   photos de toi et de ta bete.
4. Apres chaque session, entre ton meilleur tour : tape
   SEULEMENT LES CHIFFRES, l'app met les separateurs.
   Exemple : tu tapes 143256 -> l'app ecrit 1:43.256
             tu tapes 58402  -> l'app ecrit 0:58.402
   L'app garde toujours ton meilleur chrono de la session.
5. Le live timing se met a jour tout seul toutes les 5 min
   (bouton "Actualiser" pour forcer).
6. Onglet GENERAL = total cumule de toutes les sessions.
   Celui qui a couru le plus de sessions passe devant :
   impossible de gagner en sechant les sessions difficiles.
   Onglets par session = le meilleur tour de chacun.
7. Le VIOLET = meilleur tour absolu du week-end. La ligne
   bleutee = toi.

POUR LES SPECTATEURS
----------------------------------
Lien de l'app > "Mode spectateur" > code du GP.
Ils voient le live timing ET la grille de depart
(bouton "Voir la grille de depart") : toutes les cartes
pilotes, photos, ecuries, numeros et meilleurs tours.
Ils ne peuvent rien modifier.

LA GRILLE DE DEPART
----------------------------------
Depuis le live timing, "Voir la grille de depart" affiche
la carte de chaque pilote inscrit, classee par numero.
Accessible aux pilotes comme aux spectateurs. Seule ta
propre carte reste modifiable, via "Modifier ma carte".

L'HISTORIQUE
----------------------------------
Sur l'accueil, "Historique des grands prix" liste tous les
week-ends termines : pour chacun, le meilleur tour de
chaque pilote, du plus rapide au plus lent.

BON A SAVOIR
----------------------------------
- Ton palmares (records par circuit) est permanent, d'une
  annee a l'autre. Les donnees des invites ne durent que le
  temps du week-end.
- Ton chrono par session = ton MEILLEUR tour. Si tu entres
  un temps plus lent, l'app garde l'ancien.
- Saisie des temps : que des chiffres, dans l'ordre
  minute / secondes / millisiemes. Pas de deux-points ni
  de point a taper. Une erreur ? Efface et retape.

Si tu recharges la page en plein week-end, l'app te
remet exactement ou tu etais.

Bonne piste, et que le meilleur gagne.
